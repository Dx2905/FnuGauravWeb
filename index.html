Research

My current research interests lie at the intersection of machine learning, uncertainty quantification, and human–AI interaction. I focus on how calibrated models, explanations, and visualizations can help non-experts make safer decisions with AI systems, especially in high-stakes settings like healthcare.

Methodologically, I work with probability calibration (reliability curves, Brier scores), bootstrap-based uncertainty for individual predictions, and explainability techniques such as SHAP to analyze feature contributions. On the interface side, I design and prototype interactive dashboards that surface these signals in a way that is understandable and actionable for clinicians and other end users.

Long term, I want to build uncertainty-aware decision support tools that make model confidence, error modes, and limitations visible instead of hidden—so people can collaborate with AI systems without over-trusting them.

Research Themes

  • Uncertainty & Calibration

    Studying how probability calibration, confidence scores, and error analysis can be communicated in a way that matches human expectations and decision needs.

  • Explainability & Model Behavior

    Using SHAP and other attribution methods to reveal which features drive predictions, how these relate to uncertainty, and where high-confidence errors occur.

  • Clinical AI Dashboards

    Prototyping interactive dashboards that combine risk predictions, confidence intervals, and explanations to support clinician decision-making.

  • Human–AI Decision Making

    Exploring how different visual encodings of uncertainty and explanations affect user trust, perceived reliability, and attention to edge cases.

Research-Style Projects

  1. Clinical Uncertainty & Explainability Dashboard (Heart Disease & Diabetes)

    Ongoing independent project

    Built calibrated logistic regression and random forest models for heart disease and diabetes risk, then designed an interactive dashboard that surfaces predicted risk, reliability curves, and bootstrap uncertainty for individual patients. Integrated SHAP explanations to show how each feature contributes to a given prediction and to highlight high-confidence false positives/negatives.

    Focus: uncertainty visualization, calibration, SHAP, clinical decision support.
    Artifacts: mini-paper draft, Streamlit prototype, model artifacts.

  2. Machine Learning for Breast Cancer Diagnosis – Explainability & Error Analysis

    Course project extended into a research-style pipeline

    Developed multiple classifiers for breast cancer diagnosis (logistic regression, SVM, random forest) and extended the project with MLflow tracking, SHAP explanations, and monitoring. Analyzed which feature patterns led to confident but incorrect predictions and how model choice affected calibration and interpretability.

    Focus: model comparison, SHAP, calibration, reliability vs. accuracy.
    Artifacts: code repository, experiment logs, SHAP visualizations.

  3. AI Cognitive Classification (Bloom’s Taxonomy NLP)

    Educational NLP and cognitive skill modeling

    Designed a text classification pipeline that assigns exam questions to Bloom’s Taxonomy levels using TF-IDF, word embeddings, and classical ML models. Reflected on how classification uncertainty and decision boundaries could be communicated back to educators through visual summaries and explanation techniques.

    Focus: NLP, educational data, model uncertainty, explainability.
    Artifacts: GitHub project, Medium article, planned SHAP-based extensions.

  4. Toxic Online Behavior Detection with Classic ML & Transformers

    Applied ML + early exploration of model behavior

    Implemented both classic ML models and deep learning architectures (CNN/LSTM, transformer-based zero-shot models) for multi-label toxic comment classification. Compared how different models handle borderline cases and ambiguity, and identified opportunities to surface uncertainty and explanation cues to moderators.

    Focus: NLP, robustness on edge cases, trust and safety.
